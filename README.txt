При нажатии пробела создается новая нода, она проверяется на уникальность по своему числу и добавляется в контейнер с классом level-container.
Также каждый див с классом level-container соответствует уровню бинарного дерева, уровень вычисляется при создании нового узла,
с помощью специальной функции, которая вычисляет самый нижний уровень дерева.
